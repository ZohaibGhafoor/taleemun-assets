# taleemun-assets

Public brand assets for the Taleemun platform tenants, served via jsDelivr:
  https://cdn.jsdelivr.net/gh/ZohaibGhafoor/taleemun-assets@<tag>/<path>

Tags are immutable on jsDelivr — every asset change gets a new tag and updated
tenant config. Canonical sources + provenance: the (private) taleemun-platform repo.
